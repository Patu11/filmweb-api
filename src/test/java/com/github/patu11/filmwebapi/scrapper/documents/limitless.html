Opis

 (1)

Brian Finch (Jake McDorman) postanawia przetestować eksperymentalny lek NZT, umożliwiający w pełni wykorzystywać swój umysł. Mężczyzna zostaje zmuszony do systematycznego zażywania go, aby pomagać FBI w rozwiązywaniu skomplikowanych zbrodni. Wkrótce odkrywa, że spożywanie go ma także skutki uboczne.
